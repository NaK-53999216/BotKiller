BotKiller Protocol

Unstoppable AI: no central servers, no corporate control.

BotKiller is a decentralized P2P network designed to run AI beyond censorship and shutdown risk. Workers provide compute. Auditors verify results through staking. Users pay in credits. As adoption grows, BOKI (the network token) captures the upside of demand and security.

Workers = compute
Auditors = verification
Staking = trust cost
Credits = user payments
BOKI = network value

Workers

Anyone can contribute compute. Workers execute tasks and earn from usage demand.

Auditors

Independent verifiers stake BOKI to validate outputs and create an economic cost for dishonesty.

Credits → BOKI

Users pay in credits. Network growth and verification pressure increase BOKI’s utility and value.
